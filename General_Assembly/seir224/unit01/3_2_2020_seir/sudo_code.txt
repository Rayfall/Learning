Challenge 1
Given a string, write the pseudocode function to check if it is palindrome or not. A string is said to be palindrome if reverse of the string is same as string. For example, “radar” is palindrome, but “radix” is not palindrome.
=============================
SET string equal to " "
SET temporaryString equal to " "

OUTPUT "Please enter a palindrome string"
INPUT string

FOR each character up to the string length
    MOVE last character subtracted by the iteration count in string to temporaryString
ENDFOR

IF string and temporaryString are the same string
    PRINT "This string is a palindrome"
ELSE
    PRINT "This string is not a palindrome"
ENDIF

=============================

FUNCTION VowelCount(str) 
    SET phraseArray the length of the str
    SET count equal to zero

    FOR each character, starting at zero, in the length of the str
        SET each phraseArray space to the corresponding character in str
    ENDFOR

    FOR each space, starting at zero, in the length of phraseArray
        IF phraseArray equals an a, e, i, o, or u
            add +1 to count
        ELSE
            continue
        ENDIF
    ENDFOR

    return count
